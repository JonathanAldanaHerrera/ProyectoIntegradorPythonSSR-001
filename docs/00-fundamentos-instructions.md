🧭 FASE 1 - Fundamentos y elección de framework (sintetiza L1) — 3h
Contexto: Antes de escribir un solo widget, decides con qué construirás. Elegir mal el framework GUI cuesta semanas de retrabajo.

Actividades detalladas:

1. **Panorama de frameworks**: Matriz comparativa Tkinter vs PyQt6 vs Kivy vs Flet en docs/00-fundamentos.md (curva de aprendizaje, licencia, look nativo, empaquetado, comunidad).
2. **Decisión justificada**: Elige uno para el proyecto con una justificación de ≥150 palabras ligada al caso LogiTrack (escritorio de mostrador, offline-friendly).
3. **Esqueleto de ventana**: Crea la ventana raíz mínima que abre, muestra el título “LogiTrack Desktop” y cierra limpio. Es tu “hola mundo” visual.
4. **Estructura del repo inicial**: logitrack/, docs/, tests/, requirements.txt, README.md.

**Entregable de la fase:**
 *docs/00-fundamentos.md* con la matriz y la decisión, más el esqueleto de ventana ejecutable (python -m logitrack abre una ventana vacía).

**Bonus +30 XP**: Un segundo prototipo mínimo en otro framework para respaldar tu decisión con evidencia real, no solo teoría.
